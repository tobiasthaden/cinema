media.js

A modern media player for the 80% use case.

Get Started

Simplicity First

Media.js provides a simple and clean API to build and control your media.

Modular with ES6+

There is no bundle shipped just extendable modules with common defaults.

Lightweight

6.2kb gzip and dependency free. So pull in Media.js is a no-brainer.
